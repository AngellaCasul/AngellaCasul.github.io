## Project Overview

This portfolio website is designed with a clean and professional layout using minimal colors (white and navy blue). It highlights my background as an IT student, technical skills, and academic projects in a structured and easy-to-navigate format.

The website is fully responsive and includes smooth scrolling, hover effects, and scroll-based animations to enhance user experience while maintaining a business-oriented design.

## Features

- Responsive, one-page portfolio layout
- Smooth scroll navigation
- Animated section reveals on scroll
- Interactive project cards with GitHub links
- Categorized skill badges
- Certification display with badge and external link
- Clean and professional UI suitable for resumes and academic submissions
